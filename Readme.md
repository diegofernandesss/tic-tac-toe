# Tic-tac-toe - Jogo da Velha Online Multiplayer

Jogo da velha online multiplayer criado em Python + Flask + Javascript

## Instalação

Gerando uma máquina virtual por meio do virtual env

```bash
python -m venv env
```

Executar o main.py

```bash
python main.py
```
